String Object

String Methods (behaviors)

  • Get the length of the String
  • Determine if the String contains a certain value (or not)
  • Convert the String to lowercase or uppercase

An example


String className = "Intro to Java";
System.out.println("Length: " + className.length());
System.out.println("Contains Java: " + className.contains("Java"));
System.out.println("It's all lower " + className.toLowerCase());
