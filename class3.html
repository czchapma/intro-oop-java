Member Variables

An Object has one or more member variables that describe its state.

public class MotorVehicle {
boolean isEngineRunning;
int currentSpeed;
int maxNumberPassengers;
int numCurrentPassengers;
}

Defining Member Variables


		public class MotorVehicle {
		   int currentSpeed;
		   boolean isEngineRunning;
		}

Defining a Constructor

A constructor is a special method that is invoked
when you create a new instance of your object.

Code It!


public class MotorVehicle {
  int currentSpeed;
  boolean isEngineRunning;

  public MotorVehicle() {
    currentSpeed = 0;
    isEngineRunning = false;
  }
}

Defining Methods

Methods represent the behavior of the Object.

Method format

[Access Level] [Return Type] methodName(arguments)

public boolean isInClass(String student)

public int getNumStudentsInClass()

public void setClassHasStarted(boolean classHasStarted)

We've already seen this with the public modifiers on our class declaration and method declaration

Access Modifiers

Public members can be accessed by all other classes.

Private members can only be accessed within its own class.

Using Objects

Once you have created your object, you can call its different methods: 


		nameOfInstance.methodName();

Similarly, you can access member variables using this syntax: 


		nameOfInstance.dataMember;

Static vs. Instance

When a method is declared static,
it is shared by all instances of the class.

In contrast, the type of methods we've been using
have been unique to each instance.

Inheritance and Subclasses

A class can derive from another class

A class that is derived from another class is called a subclass

Subclasses inherit fields and methods from their parent class

Why would you want it? Inheritance

  • New class is created from an existing class
  • Reuse code
  • Can be used to create a hierarchical class structure

Subclassing

Add the extends keyword and the name of the parent class (MotorVehicle)


	          public class Car extends MotorVehicle {
	          ...
	          }

	          public class Bus extends MotorVehicle {
	            ...
	          }

What can you do with a Subclass?

You can write a subclass constructor that
invokes the constructor of the superclass

You can declare new fields in the subclass
that aren't in the parent class

Inherited fields can be used directly

A subclass can't inherit private members
