Main Method

In Java, every application contains a main method that acts as an entry point for the application.

Comments

Comments let you write notes in the code.

They're really useful when you're collaborating with other developers
or when you go back and look at your own work.

They can be formatted in two ways:

  1. A single line that starts with //
  2. One or more lines between /* and */

Equality and Relationship Operators

These operators evaluate to true or false

== Equal
!= Not Equal
< Less than
> Greater than
<= Less than or Equal to
>= Greater than or Equal to

Example


double costOfSubway = 2.25;
double amountOnCard = 3.50;
boolean canPay = amountOnCard >= costOfSubway;
System.out.println("You have enough for the subway: " +  canPay);

Output:


          You have enough for the subway: true

Example


int numPassegners = 3;
int maxNumberPassengers = 4;
boolean isFull = (numberPassengers == maxNumberPassengers);
System.out.println("The car is Full: " + isFull);

Output:


          The car is full: false

Conditional Operators

&& And
|| Or

We can check if two (x and y) values are true


int x = 5;
int y = -2;
boolean positiveNumbers = x > 0 && y > 0;

Conditional Operators


boolean isOdd = true;
boolean primeNumber = x == 2 || isOdd;

Precedence

Our previous example: 

  positiveNumbers = x > 0 && y > 0;

According to the precedence chart, evaluates:

x > 0, y > 0 relational
&& logical and
= assignment
